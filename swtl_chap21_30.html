Chapter 21 Concept: Changing Case.

How to change the strings to upper-case or lower-case. It is useful to compare a user input with an existing array because it should make the case of the letters all the same.

Syntax

Changing a string to lower-case.

stringVariable = 
  stringVariable.toLowerCase();

Changing a string to upper-case.

stringVariable = 
  stringVariable.toUpperCase();

Example

Changing a string to lower-case.

var deerName = "Mark";
deerName = deerName.toLowerCase();

Changing a string to lower-case.

var deerName = "Mark";
deerName = deerName.toUpperCase();

Chapter 22 Concept: Measuring Length and Extracting Parts.

This chapter reiterates the concept of the slice() for copying an array. This chapter focuses on extracting parts of a string using the slice() method.

Syntax

Using the slice method to extract part of string and assing its value a new variable.

var stringVariable = "string";
var newStringVariable = 
  stringVariable.slice(starting index, 
                      ending index + 1)

These are a few points of remember:

  • A string is an array of characters.
  • To loop through a string, stringName.length is used like with an array.
  • Some use of this concept is to check for errors in strings or to capitalize parts of a string.

Example

The example below will take a string and capitalize the first letter.

var deerName = "mark";
var firstDeerChar = deerName.slice(0,1);
var otherDeerChar = deerName.slice(1);
firstDeerChar = 
  firstDeerChar.toUpperCase();
otherDeerChar = 
  otherDeerChar.toLowerCase();
console.log(firstDeerChar+otherDeerChar);

Using the this concept, we Vegan Deer will check to see if your name has a letter "H" anywhere. We will also capitalize the first letter.

Chapter 25 Concepts: Strings Replacement Characters.

Using replace() to replace parts of a string.   

Syntax

Replacing just one instance of the segment.

var text = "Some string";
var newText = 
  text.replace("text to be replaced",
  "text to replace");

Replacing every instance of the segment in the text.

var text = "Some string";
var newText = 
  text.replace(/text to be replaced/g,
  "text to replace");

Example

Replacing "Dear" by "Deer".

var deerGreeting = 
  "Welcome to The Vegan Deer!";
var newGreeting = 
  deerGreeting.replace("Dear", "Deer");

Replacing every instance of "Dear" by "Deer".

var aboutDeer =
  "The Vegan Dear is inspired by Dear";
var newAboutDeer = 
  aboutDeer.replace(/Dear/g, "Deer");

Chapter 26 Concepts: Rounding Numbers.

How to round a decimal number to the nearest integer.

Syntax

round() rounds up when the decimal is 0.5. 

var someNumber = 
  Math.round(decimal to be rounded);

ceil() rounds up all decimals. 

var someNumber = 
  Math.ceil(decimal to be rounded);

floor() rounds down all decimals. 

var someNumber =
  Math.ceil(decimal to rounded);

Example

Examples of all three methods.

var deerTip = Math.round(8.59);
var deerTip = Math.ceil(8.59);
var deerTip = Math.floor(8.59);

Chapter 27 Concepts: Generating Random Numbers .

Using random() to generate random numbers or pseudorandom numbers.

Syntax 

var someVar = Math.random();

Points to remember:

  • Math.random() generates a random decimal up to 16 places.
  • The decimal can be multiplied by one hundred quadrillion to generate trillions of random numbers.

Syntax to generate an x amount of random numbers 

var randomNum = (Math.random() * x) + 1;
randomNum = Math.floor(randomNum);

Example

Generate 6 random numbers for our burger price.

var burgerPrice = (Math.random() * 6) + 1;
burgerPrice = Math.floor(burgerPrice);

Chapter 28 Concepts: Converting Strings to Integers and Decimals .

How to convert a numerical string to a number.

Syntax

Using parseInt() to convert a numerical string to a number data type. This method just lops off any decimal places.

var someString = "a number";
var number = parseInt(someString);

Using parseFloat() to conver a decimal string to a decimal number type to reserve the decimal places.

var someFloat = "a decimal";
var newFloat = "parseFloat(someFloat);

Example

Converting a prompt() user input to a number.

var deerOrder = 
  prompt("How many orders?");
var newDeerOrderNum = parseInt(deerOrder);

Converting a prompt() user input to a decimal.

var deerTip = 
  prompt("Enter tip amount.");
var newDeerTip = parseFloat(deerTip);

Chapter 29 Concepts: Converting Strings to Numbers and Numbers to Strings .

Using Number() method and toString() method to convert strings to numbers and numbers to strings respectively.

Syntax

Converting strings to numbers using Number(). This method reserves all decimal places.

var someString = "a number";

var someNum = a number;

Chapter 30 Concepts: Controlling the Length of Decimals.

Using toFixed() to control the length of decimals.

Syntax 

var someDecimal = a decimal; 
var newDecimal = 
  someDecimal.toFixed(number);

A few points to remember:

  • The number inside the parentheses represents the length of the decimal.
  • Same as rounding, to control which way the browser round when the decimal is 0.5, you have to make the '5' change to a 6 to ensure it will always round up.
  • Leave the parentheses blank for no decimal.

Example

Fixing the decimal to 2 places.

var deerTotal = 23.565;
var saleTotal = deerTotal.toFixed(2);
