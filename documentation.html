Introduction

Loops are very important in Javascript. They execute the same block of code again and again, as long as a condition is met, to prevent having to write the same code over and over again if that code needs to be repeated many times. Imagine having to write a code one thousand times to execute the same operation on different parameters.

Below are five different types of loops in Javascript:

  • While Loop
  • Do...While Loop
  • For Loop
  • For...In Loop
  • For...Of Loop

While Loop

The While Loop will run as long as the condition defined for its operation remains true. In the example below, the condition is updated inside the code block with the statement i++.

Example: 

var i = 1;
while(i < 5) {
  document.write("Welcome to The Vegan Deer!");
  i++;
}

Do...While Loop

This loop is similar to the While Loop. The only difference is this loop will run through the code at least one time regardless of the condition. After that, it will run as long as the condition remains true. Notice that the line while(i < 5) comes at the very end of the loop. This means the condition is read only after the loop has run once.

Example:

var i = 1;
do {
  document.write("Welcome to The Vegan Deer!");
  i++;
}while(i < 5)

For Loop

This loop will run based on a counter. When the counter reaches its limit, the loops stops. Inside the parentheses are the conditions for the loop. Part one is the a initialization of the counter. Part two is the condition to limit the counter. Part three is the updating of the counter.

Example:

for(var i=0; i<5; i++) {
  document.write("Welcome to The Vegan Deer!");
}

For...In Loop

This loop is used to run through each property of an object or each items of an array. We would usually use this loop if we need to do something to each property such as getting or setting it.

Example:

/*We assume we have an object called deerProfile with some properties to use this loop. */

for(var prop in deerProfile) {
  document.write("<p>" + prop + "</p><br>"); 
}

For...Of Loop

This loop is used for iterable objects like array. This loop is similar to For...In loop, but it was introduced in EC6. It's used to loop through an array.

Example:

// We have an array below

let deerName = ["John", "Christy", "Sam", "Josh"];

//Iterating over this array using For...Of Loop

for(let name of deerName) {
  document.write("name");
}
