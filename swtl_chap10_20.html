Chapter 10 Concept: If Statements.

If statements are used to make decisions based on the conditions given inside the statement.

The syntax for if statements:

Example

Chapter 11 Concept: Comparison Operators.

The Comparison Operators are used to compare two things and return a value of true or false. They are usually used to name the condition for an if statement, but they can be used anywhere in the program if needed.

The Operators

  • === : Equality :compare if two things are equal
  • !== : Not equal :compare if thwo things are not equal
  • < : Less than :compare if one is less than another
  • > : Greater than : compare if one is greater than another
  • <= : Less than or equal to:compare if one is less than or equal to another
  • >= : Greater than or equal to:compare if one is greater than or equal to another

Syntax

We can use these operators to compare numbers, strings, variable, math expressions, or a mixture of these.

if (thing to be compared comparison 
  operator thing to be compared) 
    {Code to be executed}

Example 

var x = 4;
if ( x < 5) {
 alert("The number is less than 5");
}

Chapter 12 Concept: If/Else and Else If Statements.

The If/Else statement allows the programs to take another path. The Else If statement lets the program tests for another condition when all other condition has failed and allows the program to take another path of execution.

Syntax

If/Else Stament: 

if (condition) {Code to be executed} 
else {Code to be executed}

Else If Statment: 

if (condition) {Code to be executed} 
else if (condition) {Code to be executed}

Example

An If/Else statement: 

if ( x < 5) {
alert("The number is less than 5);
} else {
alert("The number is greater than 5");
}

An Else If statement: 

if ( x < 5) {
  alert("The number is less than 5");
} else if (x === 5) {
  alert("The number is 5");
}

Chapter 13 Concept: Testing Sets of Conditions.

We can test for more than one condition in one If statement by using the && and || operators which mean AND and OR respectively.          

Syntax

Using && operator: Both conditions have to be met for the code to be executed.

if (condition && second codition) 
  {Code to be executed}

Using || operator: Only one condition has to be met for the code to be executed.

if (condition || second condition) 
  {Code to be executed}

When testing for more than two conditions, parenthesis pairs should be used to give the testing operation a proper order.

if (condition || (second condition && 
  third condition)) 
  {Code to be executed}
if ((condition || second codition) 
  && third condition) 
  {Code to be executed}

Example

Using the && operator: 

if ( (4 < 5) && (5 === 5)) {
  alert("They both work!");
}

Using the || operators: 

if ( (4 > 5) || (5 < 6)){
  alert("Only one is true,");
  alert(" but it still works."); 
}

Chapter 15 Concept: Arrays.

An Array is a variable that holds a list of values.

Syntax

An Array of Numbers.

var variableName = [ a number, a number, 
  a number, ...];

An Array of Strings.

var variableName = ["a string", 
  "a string", "a string", ...];

An Array of Mixed Data Types.

var variableName = [ "a string", a number,
   a variable, an object, ...];

To access an array item, use the array index which starts at 0.

arrayName[index number];
  • The variable names for arrays use the same variable name standards.
  • The array automatically gives index numbers to array items.

Example

An Array of numbers with 3 items.

var deerNum = [ 2, 3, 4];

An Array of strings with 3 items.

var deerName = ["Bambi", "Symba", "Nemo"];

An Array of mixed data types.

var deerMix = [ "Bamni", " is", 5, 
  " years old"];

To access Symba item in deerName, use the array index 1.

deerName[1];

Chapter 18 Concept: For Loops.

A For Loop is used to run something over and over again until a counter reachese its set limit. It allows for codes to repeat without having write many many lines of codes.

Syntax

for (numVariable = 0; 
  numVariable comparison operator 
  the times the code should run; 
  numVariable += count value) 
  {Code to be executed}

A few points about For Loop.

  • There should be a counter that sets a limit of the number of times the code should run.
  • In the code above, numVariable is a variable declared to be the counter.
  • The counter is usually set at 0 to start the count. But it can be set to any value.
  • Comparison operators are used to set the repetition limit the code should run.
  • The count value is the increment of the numVariable set for the loop.

Example

The example below will log "Hello" to the console 5 times.

var i;
for (i = 0; i < 5; i++){
console.log("Hello"); }

Chapter 20 Concept: Nested For Loops.

For Loops can be nested inside another For Loops as many times as infimity. The inner loop will run its entire cycle as many times as the outer loops dictates.

Syntax

The syntax below looks simple, but it is nothing as simple as it looks.

for (counter) {
for (counter) {Code to be executed}}

Example

The example below will print out 25 lines of "Lunch is served".

var i;
var j;
for (i = 0; i < 5; i++) {
  for (j = 0; j < 5; j++) {
    console.log("Lunch is served.");} }
